# 📝 SKILL: MODULE_DOCUMENTATION

Kỹ năng này đảm bảo tính cập nhật của tài liệu song hành cùng mã nguồn.

## Triggers
- "Tạo module mới [Tên Module]..."
- "Cập nhật tài liệu cho module..."
- "Skill 5 [Tên Module]"
- Lệnh `/doc` hoặc `/update-doc`

## Workflow
1. **Initialize/Read:** Tìm tài liệu `docs/modules/[ten_module].md`.
2. **Documenting:** Business logic, Controller, Model, View, DB Table.
3. **Changelog:** Cập nhật ngay lập tức các thay đổi.
4. **Optimization:** Gợi ý hướng tối ưu trong tương lai.
