# 🌿 SKILL: GIT_MASTER

Kỹ năng này chịu trách nhiệm cho các thao tác Git và quản lý phiên bản.

## Triggers
- "đẩy lên git"
- "Viết commit message cho..."
- Lệnh `/commit`

## Workflow
1. **Định dạng Conventional Commits:** 
   - `feat:` (Tính năng mới)
   - `fix:` (Sửa lỗi)
   - `docs:` (Tài liệu)
   - `style:` (Format code)
   - `refactor:` (Viết lại code)
   - `test:` (Thêm test)
   - `chore:` (Bảo trì)
2. **Commit Message:** Luôn đính kèm tên Module hoặc mô tả ngắn gọn.
3. **Push:** Sau khi commit, thực hiện đẩy code lên nhánh `main` (hoặc nhánh được chỉ định).
