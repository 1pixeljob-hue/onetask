# 🧪 SKILL: TEST_COVERAGE

Kỹ năng này đảm bảo tính ổn định và phòng tránh hồi quy (regression) cho mã nguồn.

## Triggers
- "Viết test cho file này..."
- "Kiểm tra tính đúng đắn của logic..."
- Lệnh `/test`

## Workflow
1. **Analyze:** Phân tích Controller/Model/Service/JS Function.
2. **Edge Cases:** Xác định các trường hợp biên, rỗng, sai kiểu, SQLi/XSS.
3. **Generate Code:** Sinh file Test (PHPUnit cho PHP, Jest/Mocha cho JS) kèm assertions chuẩn.
4. **Mocking:** Mô phỏng dữ liệu đầy đủ.
